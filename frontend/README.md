# MTB GUI

This is the GUI for the MONOCLE project.

## Services

### Internal services

`mtb-gui`

### External services

`Keycloak` as an identity and access manager with openID protocol

## Installation

#### Environment Variables
If you want to run the GUI without docker,
please copy the file [config.js.example](./../config.js.example) to `public/config.js`.
